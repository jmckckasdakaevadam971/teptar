# 🌳 Vorhda — Брендбук логотипа

**Vorhda** (Ворх Да — «Семь Отцов») — сайт о сохранении родовой памяти чеченских тейпов.

## 📦 Состав брендбука

| Файл | Назначение | Размер |
|------|------------|--------|
| `logo-full.svg`   | Основной логотип для шапки сайта | 500×180 |
| `logo-square.svg` | Квадратная версия для соцсетей и аватарок | 400×400 |
| `favicon.svg`     | Иконка для вкладки браузера | 64×64 |
| `logo-mono.svg`   | Чёрно-белая версия для печати | 500×180 |
| `preview.html`    | Страница предпросмотра всего брендбука | — |

- **🌳 Дерево** — род, тейп, преемственность поколений
- **7 листьев** — семь поколений предков (Ворх Да)
- **⭐ Золотой центр** — настоящее поколение, хранитель памяти
- **🌿 Корни** — глубина истории, связь с прошлым
- **💚 Зелёный** — Чечня, природа, жизнь
- **💛 Золотой** — память, мудрость, ценность наследия

## 🚀 Как использовать

### Просмотр
Откройте `preview.html` в браузере — увидите весь брендбук на одной странице.

### В HTML
```html
<!-- В шапке сайта -->
<link rel="icon" type="image/svg+xml" href="/assets/logo/favicon.svg">

<!-- В header -->
<img src="/assets/logo/logo-full.svg" alt="Vorhda" height="60">
```

### В React / Next.js
```jsx
import Logo from '@/assets/logo/logo-full.svg';

<img src={Logo} alt="Vorhda — Ворх Да" />
```

## 📐 Правила использования

- ✅ Сохраняйте пропорции — масштабируйте только пропорционально
- ✅ Используйте на светлом фоне основную версию, на тёмном — квадратную
- ✅ Минимальный отступ от логотипа — высота буквы «V»
- ❌ Не меняйте цвета без согласования
- ❌ Не добавляйте тени и эффекты
- ❌ Не растягивайте и не искажайте
